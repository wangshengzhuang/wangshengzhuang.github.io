数据库测试分为3步：prepare(准备测试数据)，run(开始测试),cleanup(清除测试数据)

​ 参数解析：

​ –db-driver：用到的数据库类型

​ –mysql-host：数据库的IP

​ –mysql-port：数据库的端口

​ –mysql-socket：socket的路径

​ –mysql-user：数据库用户名

​ –mysql-password：用户密码

​ –mysql-db：数据库名字，默认为sysbench，需要提前创建创好

​ –tables：生成表的个数

​ –table-size：每个表的行数

​ –report-interval：每隔多久在屏幕打印一次信息

​ –time：压测时间

​ –threads：启动多少个线程，即模拟多少个用户
